# ROBULLST 🛍️🔥

> Site de e-commerce responsivo da marca **ROBULLST**, focado em vestuário esportivo, com integração ao WhatsApp e busca de endereço automática via CEP.
> 
---

## 📖 Sobre o Projeto

É um site de e-commerce desenvolvido para apresentar produtos, facilitar o contato com clientes e oferecer uma experiência fluida tanto em **desktop quanto em dispositivos móveis**.

O projeto conta com:
- Layout **100% responsivo**
- Integração direta com **WhatsApp** para atendimento e pedidos
- Integração com a API **ViaCEP** para preenchimento automático de endereço
- Estrutura simples e objetiva para navegação do usuário

Este projeto foi criado com foco em aprendizado, prática de front-end e simulação de um e-commerce real.

---

## 🛒 Funcionalidades

- 📱 Design responsivo (mobile, tablet e desktop)
- 🛍️ Apresentação de produtos
- 💬 Botão de contato e pedidos via **WhatsApp API**
- 📍 Busca automática de endereço usando **API ViaCEP**
- 📑 Seções institucionais (Quem Somos, Contato, etc.)

## 🔧 APIs Utilizadas

### 📲 WhatsApp API
Utilizada para permitir que o usuário entre em contato diretamente com a loja ou finalize pedidos via WhatsApp.

### 📍 ViaCEP API
Usada para buscar automaticamente endereço (rua, bairro, cidade e estado) a partir do CEP informado pelo usuário, facilitando o processo de compra.
